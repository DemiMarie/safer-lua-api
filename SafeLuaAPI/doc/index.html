emit_wrapper (name, popped, pushed, stack_in, return_type, arguments)
Generates a wrapper for API function `name`.

Parameters:

  • name string The function name.
  • popped string The number of arguments popped from the Lua stack.
  • pushed string The number of arguments pushed onto the Lua stack.
  • stack_in {string,...} A list of arguments that name stack slots used by the function.
  • return_type string The return type of the function.
  • arguments {string,...} The argument names and types for the function.

Returns:

    string The wrapper C code for the function
